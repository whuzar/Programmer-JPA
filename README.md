Prosty projekt wykonany w Spring Data JPA/Hibernate, narzędzie automatyzujące proces budowy - Maven


W celu działania na własnej bazie PostgreSQL należy zmienić w application.properties:
- nazwę bazy
- nazwa użytkownika
- hasło
